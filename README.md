# 局部水平

$$
\begin{aligned}
y_t &= \mu_t + e_t, \quad \{e_t\} \sim \text{iid } N(0, \sigma_e^2), \quad t = 1, 2, \dots, n, \\
\mu_{t+1} &= \mu_t + \eta_t, \quad \{\eta_t\} \sim \text{iid } N(0, \sigma_{\eta}^2),
\end{aligned}
$$

$$y_t$$ 稱為觀測方程，代表系統在 t 時刻的輸出值

$$\mu_t$$ 稱為狀態方程，這個不可觀測，他描述系統內在演變規律
